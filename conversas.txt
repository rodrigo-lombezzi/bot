O que é o Minecraft?
Minecraft é um jogo de mundo aberto onde os jogadores podem explorar, minerar, construir e sobreviver.

Quais são os modos de jogo?
Sobrevivência, Criativo, Hardcore, Aventura e Espectador.

Qual é o objetivo principal do Minecraft?
Não há objetivo definido, mas muitos jogadores buscam derrotar o Ender Dragon ou construir livremente.

Como salvar o jogo?
O jogo salva automaticamente periodicamente. No multiplayer, o servidor controla os saves.

O que acontece quando morremos?
Você perde seus itens (a menos que keepInventory esteja ativado) e renasce no ponto de spawn.

O que é obsidiana?
Um bloco resistente criado quando lava entra em contato com água.

Como consigo diamante?
Minerando entre as camadas Y -58 e -16 com picareta de ferro ou superior.

Para que serve a esmeralda?
É usada principalmente em trocas com villagers.

O que é redstone?
Um pó que permite criar circuitos e mecanismos no jogo.

Como faço vidro?
Cozinhe areia em uma fornalha para obter vidro.

Como fazer uma picareta?
Use 3 blocos de material (madeira, pedra, ferro etc.) e 2 gravetos.

Qual é a melhor armadura?
A armadura de netherite é a mais resistente.

Como encantar itens?
Use uma mesa de encantamentos com lápis-lazúli e níveis de XP.

Como reparo ferramentas?
Use uma bigorna com outro item igual ou mesa de ferreiro com material base.

O que é Mending?
Encantamento que usa XP para reparar automaticamente o item equipado.

O que é um Creeper?
Um mob explosivo que se aproxima silenciosamente do jogador.

Como domesticar um lobo?
Dê ossos ao lobo até aparecer corações.

Como montar um cavalo?
Clique com o botão direito para montar, depois use uma sela para controlá-lo.

Como evitar ataques de Enderman?
Evite olhar diretamente nos olhos dele.

O que são Pillagers?
Saqueadores hostis que atacam vilas e podem iniciar invasões.

Como encontro um bioma específico?
Use mapas ou comandos como /locate biome (Java Edition).

Onde acho bambu?
Em biomas de selva.

O que é o Nether?
Uma dimensão infernal com mobs perigosos e blocos únicos.

Como ir para o End?
Encontre uma fortaleza, ative o portal do End com Olhos do Ender.

Como funciona o ciclo dia/noite?
O dia dura 10 minutos e a noite também. O ciclo completo tem 20 minutos.

Como fazer uma farm de cana-de-açúcar?
Use observadores e pistões para quebrar a cana e um funil para coletar.

Como automatizar fornalhas?
Use funis para alimentar a fornalha com combustível e itens a serem cozidos.

O que é um clock de redstone?
Um circuito que envia pulsos de forma contínua.

Como fazer uma porta automática?
Use pistões, placas de pressão ou ganchos para abrir e fechar portas.

Para que serve um comparador de redstone?
Detecta estados de blocos como baús, fornalhas ou caldeirões.

Como usar comandos?
Ative os cheats no mundo e pressione T para abrir o chat.

Como mudar o modo de jogo?
Use /gamemode survival, creative, spectator, adventure ou hardcore.

Como teleportar?
Use /tp <jogador> <x> <y> <z>.

Como dar itens via comando?
Use /give <jogador> <item> [quantidade].

Como invocar mobs?
Use /summon <tipo_de_mob> <coordenadas>.

Como encontrar uma vila?
Explore o mundo ou use /locate structure village (Java Edition).

O que tem em uma Fortaleza do Nether?
Spawners de blaze, baús e varas de blaze.

Como funciona o templo do oceano?
Abriga guardiões e esponjas, e só pode ser explorado com poções de respiração aquática.

O que é uma mansão?
Grande estrutura no bioma de floresta escura com mobs Illagers.

O que são portais em ruínas?
Estruturas de nether quebradas com blocos de obsidiana e baús ao redor.

Como respirar debaixo d'água?
Use poções, capacete com Respiração ou bolhas de magma.

Como evitar dano de queda?
Use botas com Pena Leve, água ou blocos como feno ou teia de aranha.

O que são encantamentos amaldiçoados?
Maldição do Desaparecimento e da Ligação Eterna impedem drop ou remoção de itens.

Como domar um gato?
Dê peixe cru até o gato confiar em você.

Como conseguir sela?
Não pode ser craftada, apenas encontrada em baús de estruturas, pescando ou com villagers.

Como ganhar XP rápido?
Crie farms de mobs, extraia minérios ou use fornalhas para ganhar XP.

O que é lapis-lazuli?
Mineral usado para encantar itens na mesa de encantamento.

Qual o nível máximo de encantamento?
A maioria vai até nível 5. Com comandos, pode ser maior.

Como usar a bigorna?
Combine dois itens ou renomeie-os usando XP.

Como funciona o livro encantado?
Use bigorna para aplicar o encantamento no item desejado.

Como derrotar o Ender Dragon?
Destrua os cristais nos pilares e ataque o dragão quando ele pousar.

O que o Wither dropa?
Uma Estrela do Nether, usada para fazer o beacon.

Como criar o Wither?
Coloque 4 blocos de soul sand em T e 3 cabeças de esqueleto wither.

O que acontece ao matar o dragão?
Um portal para casa é aberto, junto com o ovo do dragão.

Como encontrar bastiões?
Estão no Nether, onde Piglins Brutos guardam baús com loot valioso.

Como faço uma farm de trigo, batata ou cenoura?
Plante os itens em solo arado com água ao redor, use villagers (farmer) com acesso à plantação e um funil/coletor embaixo para automatizar a colheita.

Como faço uma farm de cana-de-açúcar?
Plante em terra ao lado de água. Use observadores para detectar crescimento e pistões para quebrar a cana automaticamente. Um carrinho com funil coleta por baixo.

Como faço uma farm de abóboras e melões?
Plante o caule em um bloco e ao lado deixe um espaço livre. Use observadores para detectar o crescimento do fruto e pistões para empurrá-lo para funis.

Como faço uma farm de cacto?
Plante em areia com blocos ao lado para forçar o cacto a quebrar quando crescer. Um funil embaixo coleta o item.

Como faço uma farm de bambu?
Mesma lógica da cana-de-açúcar: plante em terra, use observadores e pistões para colher automaticamente. Use carrinho com funil para coleta.

Como faço uma farm de vacas, ovelhas e porcos?
Crie um cercado e alimente-os com trigo ou cenoura. Para automação, use lava temporária com dispensadores para abate e funis para coleta.

Como faço uma farm de galinhas?
Coloque várias galinhas em um bloco pequeno com funil embaixo. Use dispensadores para lançar ovos e gerar novas galinhas, e lava para matança automática.

Como faço uma farm de abelhas?
Coloque colmeias com flores próximas. Use dispensadores com tesouras ou frascos para coletar mel e favos. Garanta iluminação e flores por perto.

Como faço uma farm de pesca AFK?
Construa um local com água, gancho, bloco de ferro e vara de pesca com Mending. Use macro ou peso no botão direito (Java Edition). Necessária posição exata do cursor.

Como faço uma farm de ferro?
Coloque 3 villagers com camas e estações de trabalho, e um zumbi para assustá-los. Os Golems surgem e caem em lava com funis abaixo para coleta.

Como faço uma farm de ouro e piglin bartering?
Construa acima do Nether (Y > 256) com portais gigantes ou use piglins para coletar ouro automaticamente. Use dispensadores para alimentar os piglins.

Como faço uma farm de lava?
Use goteiras (dripstone) embaixo de caldeirões com lava acima para gerar lava ao longo do tempo.

Como faço uma farm de pedra infinita (cobblestone generator)?
Coloque lava de um lado e água do outro. Quando entram em contato, formam pedra. Use pistões e redstone para empurrá-las automaticamente.

Como faço uma farm de cobre?
Não é automática. O cobre precisa ser minerado em biomas de cavernas (Y47). Para automatização, use TNT dupers ou pistões para mineração em larga escala.

Como faço uma mob farm básica?
Construa uma torre escura (Y > 128), com água empurrando mobs para uma queda. Os mobs morrem por queda ou por lava e são coletados por funis.

Como faço uma farm de creepers?
Construa uma mob farm com altura de 2 blocos para permitir apenas creepers. Use tapetes e gatos para assustá-los e direcioná-los para a morte.

Como faço uma farm de blaze?
Construa em um spawner de blaze. Use pistões para empurrá-los para uma câmara de matança com lava ou espadas.

Como faço uma farm de enderman?
Construa no The End, longe da ilha principal. Use uma Endermite para atrair e prender Endermans para matá-los.

Como faço uma farm de slime?
Construa em chunk de slime (Y0–Y40). Use golems para atrair slimes para uma câmara de morte por queda ou lava.

Como faço uma farm de bruxas (witches)?
Construa em uma cabana de bruxa (pântano). Use plataformas e mecanismos para empurrá-las para uma câmara de morte com lava ou esmagamento.

Como faço uma farm de villagers e esmeraldas?
Crie vilas artificiais com villagers de profissões específicas (Farmer, Mason). Troque abóboras, pedras ou argila por esmeraldas. Curar villagers zumbis dá desconto.

Como faço uma farm de wither skeletons?
Construa plataformas no Nether, dentro das fortalezas. Use slabs para prevenir o nascimento de mobs indesejados e leve-os para uma zona de morte com golens de ferro ou tartarugas.

Como faço uma farm de estrela do Nether (Wither Boss)?
Mate manualmente o Wither ou prenda-o em bedrock (ex: no teto do Nether). Automatizar requer cuidados para não destruir o ambiente.

Como faço uma farm de raid (invasão)?
Provoque uma raid com efeito de Mau Presságio. Leve-os para uma armadilha e use água para empurrá-los para uma zona de morte, obtendo totens e esmeraldas.

Como faço uma farm de shulker shells?
No End, após o update 1.17, shulkers podem se reproduzir quando atingidos por uma bala de outro shulker. Use transporte e pistões para construir uma farm e matá-los com Looting III.

Como faço uma farm de sculk XP?

Como faço uma farm de madeira?
